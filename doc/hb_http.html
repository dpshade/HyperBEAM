      accept_to_codec(TABMReq, Opts)

      Calculate the codec name to use for a reply given its initiating Cowboy request, the parsed TABM request, and the response message. The precidence order for finding the codec is:

      1. The accept-codec field in the message
      2. The accept field in the request headers
      3. The default codec Options can be specified in mime-type format (application/*) or in AO device format (device@1.0).
